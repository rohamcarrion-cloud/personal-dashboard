# OAuth Setup Guide

This guide walks you through setting up OAuth credentials for all supported social media platforms. Once configured, users can securely connect their social accounts through the Command Center.

## Overview

OAuth allows users to authorize the application to post on their behalf without sharing passwords. Each platform requires:
1. A registered application in their developer console
2. OAuth credentials (Client ID, Client Secret)
3. A redirect URI configured in the platform's settings
